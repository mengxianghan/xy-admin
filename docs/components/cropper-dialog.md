# CropperDialog 图片裁剪弹窗

## API

### Props

| 名称          | 说明                           | 类型       | 默认值 |
|-------------|------------------------------|----------|-----|
| aspectRatio | 比例，例如：`0` `1/1` `4/3` `16/9` | `number` | `0` |
| quality     | 输出质量，取值范围：`0-1`              | `number` | `1` |

### Events

| 名称     | 说明  | 返回值              |
|--------|-----|------------------|
| ok     | 确定  | `function(file)` |
| cancel | 取消  | -                |

### Function

| 名称         | 说明   | 参数            |
|------------|------|---------------|
| handleOpen | 打开弹窗 | `src: string` |
