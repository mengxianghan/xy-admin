### Props

| 名称    | 说明                                   | 类型     | 默认值 |
| ------- | -------------------------------------- | -------- | ------ |
| urls    | url 数组，支持 `image/*` `.mp3` `.mp4` | `array`  | `-`    |
| current | 预览起始位置索引                       | `number` | `0`    |
